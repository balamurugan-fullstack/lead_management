# Lead Management System - Frontend

React + TypeScript + Vite frontend for the Lead Management System.

## Prerequisites

- Node.js 20+
- npm

## Installation

```bash
git clone <repo-url>
cd frontEnd
npm install
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:3000/api
```

## Run Development Server

```bash
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
